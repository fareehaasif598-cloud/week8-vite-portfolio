# Week 8 Vite Portfolio

This project uses three JavaScript modules to organize my portfolio code. The `src/projects.js` file stores my project data and displays the project cards, the `src/api.js` file gets my GitHub repositories from the GitHub API and creates repo cards, and the `src/main.js` file imports those modules and runs the code when the page loads.
